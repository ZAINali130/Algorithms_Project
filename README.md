# Student Management System

This project is a C# application developed as an assignment for the Algorithms course. It implements a **Doubly Linked List** to manage student data and includes a **Recursive Search** functionality.

## Main Features:
* Doubly Linked List (Add Front/Back).
* Sorting by Name or Scores.
* Recursive Search by Score.
